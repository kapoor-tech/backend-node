# backend-node

## Start Server
`$ yarn start:dev`

## Docker Mysql

`$ docker-compose up`

## Users

[GET] All Users
http://localhost:4000/users

[POST] Create User
http://localhost:4000/users

[PUT] Update User
http://localhost:4000/users/id

[DELETE] Remove User
http://localhost:4000/users/id

[GET] Find User
http://localhost:4000/users/id
